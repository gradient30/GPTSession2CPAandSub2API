格式说明

本页面仅做格式转换，不能绕过 Codex OAuth 添加手机号 / 手机绑定验证。ChatGPT Web 会话通常不含 `refresh_token`；缺少真实 `id_token` 时，CPA、Cockpit、Codex 和 AxonHub 会构造 Codex 可解析的占位 JWT claims（含 `id_token_synthetic` 标记）。Codex 和 Codex-Manager 会保留空 `refresh_token`；AxonHub 缺少真实 `refresh_token` 时会写入占位值。

安全处理

  • 所有解析在浏览器本地完成
  • 不向任何服务器上传 token
  • 不写入持久化存储（除主题偏好）
  • 转换完成后建议立即清空输入
JSON 含 accessToken / sessionToken，属于敏感凭证，请勿外传。

使用限制

OpenAI 已限制通过 Web 会话转换导入来跳过 Codex OAuth 添加手机号 / 手机绑定验证；本工具仅做格式转换。

处理真实 token 时建议本地打开本页面，勿使用不可信的在线镜像。
