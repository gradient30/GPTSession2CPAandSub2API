Session 数据从这里获取

先在浏览器登录 ChatGPT，然后打开 https://chatgpt.com/api/auth/session ，复制页面显示的整段 JSON，粘贴到下方输入框。

OpenAI 已限制通过 Web session 转换导入来跳过 Codex OAuth add phone / 手机绑定验证；本页面仅用于格式转换。

这段 JSON 包含 accessToken 和 sessionToken，等同敏感登录凭证，不要发给别人。处理真实 token 时建议本地打开本页面，勿使用不可信的在线镜像。
